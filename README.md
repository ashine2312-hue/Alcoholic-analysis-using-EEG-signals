Alcoholism Detection Using EEG

Alcohol is a depressant that affects the central nervous system and can lead to serious mental, behavioral, and physical health issues. Early and accurate detection of alcoholism is important, but reliable standard methods are limited.

This project proposes a method to detect alcoholism using EEG data by combining deep learning and machine learning techniques. A CNN model is used for feature extraction, followed by an SVM classifier with cross-validation for classification.

The model achieves 96.01% accuracy with deep learning and 98.21% accuracy with machine learning, showing strong performance for alcoholism detection.
